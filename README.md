### **Adding New Events (events.html):**

1. Open `events.html` in Notepad/TextEdit
2. Find an existing event card (lines ~130-160)
3. Copy the entire `<div class="event-card">...</div>` section
4. Paste it below, change:

1. Open `leadership.html`
2. Find the chaplain section (~line 380)

5. Optionally upload a real photo
